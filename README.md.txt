# 💡 Generative AI – Prompt Lab using IBM Cloud (Watsonx.ai)

This project showcases the implementation of a **Generative AI Prompt Lab** using **IBM Cloud's Watsonx.ai** platform. It demonstrates how to build, deploy, and interact with foundational models using a prompt-based interface.

---

## 📌 Project Overview

This lab was completed on the IBM Cloud platform using **Watsonx.ai**, focusing on:

- Creating and managing a Prompt Lab project
- Associating Watsonx Runtime services
- Using **foundation models** for prompt-based tasks
- Uploading documents and chatting with AI
- Using both **Freeform** and **Chat** modes
- Understanding real-time text generation using IBM’s foundation models

---

## 🧠 Features Demonstrated

✅ Login and setup IBM Cloud account  
✅ Create and configure Watsonx.ai project  
✅ Use **Prompt Lab** to run custom text prompts  
✅ Upload documents (PDFs/text) and interact with them  
✅ Test models using **Freeform** and **Chat-based prompts**  
✅ Explore and chat with **foundation models**

## 📝 Steps Followed (as per PDF):

1. Login to IBM Cloud → Search for **Watsonx.ai**
2. Access **GenAI Solutions** → Select **Prompt Lab**
3. Create new project with Free plan
4. Associate the **Watsonx.ai Runtime** service
5. Explore prompt building (Freeform & Chat modes)
6. Upload external documents and interact via chat
7. Generate outputs and test understanding
